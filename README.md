## Introduction

Package `pkg` is a collection of useful Go packages.

## Installation

### Install using `go get`

```shell
go get github.com/axiomhq/pkg/...
```

### Install from source

```shell
git clone https://github.com/axiomhq/pkg.git
cd pkg
make # Run linters, sanitizers and test suits
```
